# Upperbody Tracker-RaspberryPi

This script tracks the upper body of the closest person and tries to keep it in the center of the frame of the camera.

## Requirements

- OpenCV
- pigpio

## Usage

```bash
sudo pigpiod
python3 upperbody.py
```
